Alternate Default Theme

Sometimes I think it's age, other times I blame it on phones and sometimes I blame it on typographers making all those lovely typefaces. All in all, lately text has been getting larger on the web and I'm mostly happy about it.

So, I made some edits to the default Indexhibit format increasing the size of the type. And, also adding ample amounts of margin everywhere. I'm using a modified version of it for my own website.

You can download Alt Default here if you wish. It's for Indexhibit v2.1 (released yesterday) but I think it will work on version 2.0. You really should update though.

I would love to know if you are using it - show me!

I have more themes, formats and plugins coming soon.
